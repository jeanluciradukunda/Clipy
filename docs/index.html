Features

Everything you didn't know
your clipboard needed.

Clipy v2 is a complete rewrite. Every panel, every shortcut, every pixel — rebuilt from scratch in SwiftUI.

🔍

Spotlight-style Search

Fuzzy search across your entire clipboard history. Filter by text, images, links, files, or pinned items. Split-pane preview with rich content rendering.

🌈

Syntax Highlighting

Auto-detects 16+ languages — JSON, Python, Swift, JavaScript, SQL, and more. Code looks like code, not a wall of text.

👁

Image OCR

Extract text from any image clip using macOS Vision. Results are cached — run it once, see it forever. No internet required.

⚡

Smart Actions

JSON pretty-print & minify. URL tracking parameter cleanup (50+ trackers). Text transforms. Clickable URL, email, and phone badges.

📋

Snippet Picker

Browse and paste snippets from a Spotlight-style panel. Dynamic variables — %DATE% %CLIPBOARD% %UUID% — expand at paste time.

🔒

Vault Folders

Protect sensitive snippets with Touch ID or password. Vault content is excluded from search, export, and the picker until you authenticate.

📥

Clipboard Queue

Collect multiple clips in sequence, then paste them all at once — merged with any separator, or one-by-one. Like a clipboard playlist.

⌨

Keyboard-first

Two-digit quick select (type 15 for item 15). Arrow navigation. ⌘D to pin. Every shortcut is configurable.

🔄

Auto-update

Checks GitHub Releases for new versions. One click to download, install, and relaunch — no manual DMG wrangling after the first install.

🔨

Developer Mode

Hidden power tools for nerds: Stats for Nerds dashboard with radar chart, hourly & daily activity graphs, feature usage counters with JSON export. Plus TipKit management, database info, hotkey disable for side-by-side testing, and a vibing Clippy.

Help

Install & Uninstall

Clipy is signed and notarized with an Apple Developer ID. Download, install, and you're ready to go.

📦

Installation Guide

Step 1: Download the .dmg from the button above

Step 2: Double-click the .dmg to open it

Step 3: Drag Clipy to the Applications folder

Step 4: Open Clipy from Applications. It appears in your menu bar

Step 5: Grant Accessibility permission when prompted (needed to paste)

Signed and notarized with Apple Developer ID — no Terminal commands needed.

✅

Signed & Notarized

Clipy is signed with an Apple Developer ID certificate and notarized by Apple. No security warnings, no Terminal commands — just download and run.

🔄

Updating to a new version

1. Quit Clipy

2. Install the new DMG (drag to Applications, replace)

3. Launch Clipy

Accessibility permission persists across updates since the app is signed with a consistent Developer ID.

🗑

Uninstall

1. Quit Clipy from the menu bar
2. Drag Clipy from Applications to Trash
3. Remove app data (optional):

rm -rf ~/Library/Application\ Support/com.clipy-app.Clipy/
defaults delete com.clipy-app.Clipy

Also remove from System Settings → Privacy → Accessibility
